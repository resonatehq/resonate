Search schedules

query Parameters
id
string

Search schedules for matching IDs, can include wildcards.

For example:

  • "foo/*" matches all IDs starting with "foo/"
  • "*/bar" matches all IDs starting with "bar/"
  • "foo/*/bar" matches all IDs starting with "foo/" and ending with "/bar"
